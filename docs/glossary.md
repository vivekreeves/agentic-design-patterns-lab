# Glossary

- Agent: A software component that pursues goals via reasoning and actions.
- Tool Use: Calling external functions, APIs, or local utilities during reasoning.
- Reflection: Self-critique step used to improve outputs.
- Planning: Explicit decomposition of goals into ordered actions.
- Routing: Selecting model, workflow, or specialist agent based on context.
- Memory: Persisted state used to improve continuity and adaptation.
- Guardrail: Constraint or policy enforcing safe behavior.
- FinOps: Practices for optimizing AI quality, cost, and latency.
- RAG: Retrieval-Augmented Generation, grounding model outputs in data.
- HITL: Human-in-the-loop, requiring operator approval on sensitive steps.
